# RSpec
## Automação de API utilizando RSpec e Ruby

### Parar rodar as tags específicas
#### rspec . --tag teste_tag
### não rodar tag específica
#### configurar no arquivo .rspec
####  --require spec_helper
####  --tag ~nao_teste_tag
#### Quando rodar rspec ele não executa/roda o que foi configurado para não rodar
